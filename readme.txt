== Description ==

This plugin checks if CapsLock key is on and if it's it shows specific message.

== Installation ==

1. Upload 'checkcl' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Have fun
